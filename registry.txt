FIELD DEFINITIONS (per run entry)
================================================================================
RUN_ID              Unique identifier for this run
PHASE               baseline | sft | lora | qlora | gpu_sweep | quant | api | bench
STATUS              planned | running | completed | failed | abandoned
CONFIG_PATH         Relative path to configs/*.yaml used
MODEL_ID            Hugging Face model id or local path label
HARDWARE            CPU model / GPU model / CUDA version (short)
GIT_COMMIT          Optional commit hash when run was executed
ARTIFACT_DIR        experiments/<RUN_ID>/ or external URI
CHECKPOINT          Path to weights or adapter folder (if applicable)
METRICS_PATH        Path to metrics.json or equivalent
BENCHMARK_TABLE     Path to exported CSV/JSON table (if applicable)
NOTES               Free text: OOM fixes, hyperparameter quirks, failures
